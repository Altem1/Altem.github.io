Sabores de Mexico

Esta pagina fue hecha con lo basico de html y css para su frontend
Sabore de mexico es un sitio web donde se trato de recopilar
parte del significado de la comida mexicana.

En el se muestran varios platillos tipico, sus recetas donde
especificamos los ingredientes y como preparlos de una manera
corta y concisa.

Tambien se agregado un buscador dentro de la pagina "Recetas"
esta se desarrollo a base de js, jquery y un framework de busqueda.

Numeros Primos

Este es un programa hecho 100% en python.

Este programa tiene 2 funciones.

  • La primera verifica si un numero es primo o no
  • La segunda te da un numero aleatoria del 1 al 100


El proposito de ambas funciones era para un proyecto escolar
de una loteria y que adivinaran si un numero era o no primo.

Practicas en C++

Este es un repositorio en Github donde subi varias de

mis practicas hechas en la preparatoria.
Se encuentra desde el uso basico de las entradas y salidad
hasta un proyecto final el cual trataba de hacer el codigo
de un menu de un restaurante, donde el usuario llegara a
ordenar en una pantalla con 3 opciones preestablecidas.

Tenia que hacer su pedido y cuantos queria y el programa
calculaba de forma automatica el monto a pagar y preguntaba
si queria ordenar algo mas, en cualquier caso se reiniciaba.

Se usaron diferentes tipos de ciclos, condicionales y funciones
El cofigo esta comentado para que se entienda mejor.

App de notas y dibujos

Esta es una aplicacion hecha en app inventor.

Esta aplicacion tiene 2 funciones principales.
La primera es para tomar notas, donde puedes escribir
lo que quieras y guardarlo en la aplicacion.

La segunda es para dibujar, donde puedes hacer un dibujo
con diferentes colores y grosores de lapiz.

Apesar de ser creada con bloques, quedo lo suficientemente completa.
